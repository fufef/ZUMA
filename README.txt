Zuma THE GAME
Это игра зума, цель которой заключается в том, чтобы уничтожить все шарики на экране, не считая шарика игрока.
На каждом уровне шарики разного цвета перемещаются по кривой. Игрок может выстрельнуть своим шариком, чтобы тот оказался в последовательности движущихся шаров.
Подряд идущие 3 или более шаров одного цвета самоуничтожаются.
Задача игрока - стрелять шариками, чтобы все они самоуничтожились до того момента, пока они не достигнут конца кривой.
